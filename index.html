Swift

Swift를 통해 iOS 앱 개발을 진행하였고, Swift의 다양한
문법들을 정확히 이해하기위해 TIL을 작성하며 공부하였
습니다. 알고리즘을 Swift로 해결하면서 문법을 사용하
는 법을 익히는 것에 그치지않고 해당 문법의 내부 구조
를 파악하였습니다.

Git, Github

5개의 팀 프로젝트 모두 Git을 통해 버전관리를 진행하였
으며, 공동작업을 위해 Github를 사용하였습니다. Git을
통해 소스코드를 관리하는 법을 익혔으며 활발한 코드리
뷰를 통해 팀원들과 지식을 공유하였습니다.

Alamofire

Alamofire을 사용하여 HTTP 통신을 진행해본 경험이 있
습니다. InFrame, EZY 등의 프로젝트에 Alamofire를 활
용하여 서버 통신을 적용하면서 Alamofire에 대한 이해
도를 높힐 수 있었습니다.

Snapkit

진행한 모든 프로젝트에서 Snapkit을 사용하여
autolayout을 적용하였습니다. 다양한 프로젝트에
Snapkit으로 autolayout을 적용하면서 뷰의 크기와 위치
를 동적으로 지정하는 방법을 익혔습니다.

HTML, CSS

앱 개발에만 국한되지 않고 다양한 분야를 경험해보았습
니다. Doory 프로젝트에서 FrontEnd 웹 개발을 맡아 퍼
블리싱을 담당하였고 그 과정에서 웹 개발에 대한 이해
도를 높힐 수 있었습니다.

Figma

사용하기 쉬우며 보기에도 예쁜 앱을 만드는 것을 좋아
합니다. 진행한 프로젝트의 디자인 대부분을 직접 제작
하였으며 그 과정에서 Figma에 대한 이해도를 높일 수
있었습니다.
